This site made as training project, based on Django 4.0.6 with a default SQLite database and styled with Bootstrap v5.2 + Fontawesome icons.


Features implemented:

  • Categorical post display in with reduced image size and text
  • Single post view with bigger image and full text
  • Add / edit post for authenticated users using CKEditor
  • Like / dislike post
  • Comment post
  • Search in posts
  • Pagination
  • Authentication / Registration
  • Profile watch and edit pages
  • User avatars
  • Feedback form with captcha check

Additional modules used:

  • Django-simple-captcha
  • Django-debug-toolbar
  • Django-bootstrap-modal-form
  • Django-avatar
  • Django-ckeditor
  • Django-widget-tweaks
